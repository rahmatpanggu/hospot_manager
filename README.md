# Hotspot Manager 🔥

Aplikasi desktop untuk mengelola WiFi Hotspot di Windows dengan mudah.

## ✨ Fitur

- ✅ Nyalakan/Matikan Hotspot
- ✅ Ganti SSID & Password
- ✅ Lihat Perangkat Terhubung
- ✅ Auto-save Konfigurasi

## 📥 Download

Download installer terbaru: **[HotspotManager_Setup.exe](https://github.com/rahmatpanggu/hotspot_manager/releases/latest/download/HotspotManager_Setup.exe)**

## 🖥️ Persyaratan Sistem

- Windows 10 atau Windows 11 (64-bit)
- RAM: Minimal 2GB
- WiFi adapter yang mendukung hosted network

## 🎮 Cara Penggunaan

1. **Download** file installer
2. **Install** aplikasi
3. **Jalankan** sebagai Administrator (klik kanan → Run as administrator)
4. **Atur SSID & Password**
5. **Klik ON** untuk mengaktifkan hotspot

> ⚠️ **Catatan**: Aplikasi harus dijalankan sebagai Administrator!
